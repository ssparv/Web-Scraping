# Web-Scraping
Web Scraping is the process of extracting data from the website. Here web scraping is done in Python by using requests and BeautifulSoup 
modules. In this repository:

- First Web scraping is done to extraxt the Top 250 IMDB movies from website and is loaded into an Excel file using openpyxl module.
- Second web scraping is done to get collection of books from e-commerce webiste which has about 50 pages. By using pandas the extracted data 
is stored to a daframe and is uploaded to a csv file.
